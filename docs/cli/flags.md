# CLI Flags

This document lists the available command-line flags for the Gemini CLI.

## General Flags

- `--voice` (experimental): Enables a hands-free, voice-driven interaction mode.
  In its current version, this mode streams microphone audio to Gemini Live API
  and handles transcript-driven command execution. In voice mode, say `help`,
  `what can i say`, or `commands` to view available commands, and say `exit` to
  leave voice mode.
